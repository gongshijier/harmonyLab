# harmonyLab

> 基于浏览器基本 UI 特征，熟悉 ArkTs 语法，方便快速上手鸿蒙开发  
学习博客：https://bzreny5z71.feishu.cn/docx/OJ74dfqe4oHXNNxpT5Mc1Bujnjg  
***
目标  
构建完整的鸿蒙组件使用指南，包含各种组件的使用示范 并且 沉淀通用的组件  
***
核心功能  
* 频道能力
* Tab能力
* 信息流能力：数据加载组件（内存 数据库 网络）、 列表页组件    
***
基本组件  
- 频道--Tabs
- 底 Tab--Tabs
- 列表页--List
